My Work

DC Roasters


DC Roasters is a full stack e-commerce website for users to purchase coffee via a monthly subscriptions. Users signup, login, place orders and make payments.

Sportakus


Sportakus is a frontend sports aggragator that allows the user to select a sport and then select a venue displayed on a google map. The user can see the events for that choosen sports venue and ticket information from Ticket Master.

BlackJack


BlackJack Casino is a frontend blackjack game where the user plays against the dealer. The game includes score keeping and checks for winners and busts.

Memory Game


The Memory Game is a matching game that requires the user to select a card and then find its match after the card turns over. The game is customized for DigitalCrafts.

Chirp


The Chirp application is a dynamic database driven web application that is modeled after Twitter. Users can chirp messages, follow friends, like chirps, and rechirp other chirps.

Daily Planner


The Daily Planner is a frontend React JS application that allows users to schedule hourly appointments with their name and phone number Users can edit and cancel appointments.

Valet Me


Valet Me is a Full Stack cloud based parking management system. The mobile app allows user to signup, login, request there car be returned, and tip or pay the valet company. Push notification are used for notifying users when there car is ready and includes a live interactive administration panel that uses socket.io to help manage the parking lot.
